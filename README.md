Lord of the Mods by Błażej Chojnacki

The Lord of the Mods application is a game modification switcher and editor for the Electronic Arts games:
The Battle for the Middle-earth II and its extension - The Rise of the Witch-king.

The application automates defining mod files, enabling and disabling mods, reading, editing and fixing .INI files.

Current version 1.0.0
